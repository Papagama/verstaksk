# Portfolio Website

Сайт-портфолио для разработчика лендингов и сайтов.

## Структура проекта

```
portfolio/
├── css/                    # Стили
│   ├── critical.css       # Критические стили
│   ├── main.css           # Основные стили
│   ├── portfolio.css      # Стили портфолио
│   └── responsive.css     # Адаптивность
├── js/                     # JavaScript
│   ├── animations.js      # GSAP анимации
│   └── portfolio.js       # Данные и рендеринг портфолио
├── images/                 # Изображения
├── index.html             # Главная страница
├── case.html              # Шаблон кейса
└── .htaccess              # Конфигурация сервера
```

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript (ES6+)
- GSAP (анимации)
- GitHub Pages (хостинг)

## Особенности

- Адаптивный дизайн (мобильные, планшеты, десктопы)
- Оптимизация производительности (WebP, кеширование, сжатие)
- SEO-оптимизация
- Плавные анимации при скролле
- Пагинация портфолио

## Разделы сайта

1. **Hero** — Главный экран с заголовком
2. **О специалисте** — Информация о разработчике
3. **Услуги** — Список услуг
4. **Портфолио** — Кейсы с пагинацией
5. **Подход** — Этапы работы
6. **Экспертность** — Ключевые компетенции
7. **Контакты** — Форма связи

## Управление контентом

Кейсы портфолио находятся в `js/portfolio.js`:

```javascript
const portfolioData = [
  {
    id: 1,
    title: "Название проекта",
    tag: "Категория",
    description: "Описание",
    image: "images/case1.png",
    link: "https://...",
    // ... остальные поля
  }
]
```

## Деплой

Сайт автоматически публикуется через GitHub Pages:

```bash
git add .
git commit -m "Описание изменений"
git push origin main
```

Изменения появляются на сайте через 1-2 минуты.

## Производительность

- Первая отрисовка контента: <1.5s
- Время до интерактивности: <3.5s
- Оптимизированные изображения (WebP)
- Кеширование в браузере (1 год для изображений, 1 месяц для CSS/JS)
- GZIP сжатие

## Цветовая схема

- Акцент: `#FF4D00` (оранжевый)
- Фон: `#FFFFFF` (белый)
- Текст: `#000000` (черный)
- Приглушенный: `#666666` (серый)
